## 🔍 如何预览图标

### 方法1: 使用预览页面（推荐）
1. 启动开发服务器：
   ```bash
   npm run dev
   ```
2. 在浏览器中访问：
   ```
   http://localhost:3000/icon-preview.html
   ```
3. 预览页面会显示：
   - 所有8个图标的大图预览
   - 不同尺寸的显示效果（64x64、32x32、16x16）
   - 点击选择功能
   - 分为"小红书风格"和"硅谷风格"两个区域

### 方法2: 直接查看SVG文件
- 在文件管理器中打开 `public` 目录
- 双击任意 `icon-option-*.svg` 文件在浏览器中查看
